玩转电脑之windows上的好物推荐

发表于 2017-04-05 更新于 2019-04-25 分类于 工具

微软的windows操作系统着实推动了人类社会发展的一大进步，他的强大和普及有目共睹。然而这么强大的操作系统也不能万能，在此介绍几款windows上的小公举来提高效率，健康生活。

工具，为我所用才是它存在的意义嘛！

windows维护清理工具

CCleaner

博主本人从来不用安全管家，360卫士之流的软件。因为这些软件真心很流氓，动不动弹出一些不疼不痒的通知来显示自己的存在感，还时不时绑架全家桶，后台运行，强行抢占内存，导致电脑运行缓慢，无耻，流氓！所以推荐国外这款清理工具，清理系统垃圾，软件缓存，清理注册表都有不错的效果，建议使用。

支持清洁，杜绝流氓！

MSE

MSE—Microsoft Security Essentials，微软自家杀毒软件，windows 7系统可以安装，使用这个保护电脑足够了，卸掉360之流吧。windows 8系统以上，无需安装，系统已自动嵌入，叫windows defender，我电脑整天都是这样裸奔过来的，也没见发生什么。

windows截图工具

snipaste

作者十年磨一剑，绝对精品，windows上最好用的截图工具，我的大多数截图均实用该软件实现，给作者点赞。

LICEcap

很小又超级好用的GIF录屏工具，可以自定义图片大小，帧率，插入文字，而且安装包大小只有200多K，好用到令人窒息。

windows搜索工具

everything

windows上最好用的搜索工具，没有之一。everything的搜索能力比windows自带的搜索不知道要强多少倍，任何文件，即搜即到，而且大小只有不到2M，良心之作，微软你学到了吗？！

配合上windows快捷键，任何时刻按下Ctrl + Alt + S, 瞬间启动，怎样，够geek吧！好用到不可方物！

windows空格键预览文件工具

QuickLook

Mac上的空格预览功能非常实用，想要预览文件，只需选中，按下空格，就会迅速打开预览，无需打开软件加载，非常高效实用。可这么好用的功能，Windows上并没有提供，然后改变世界的程序员站出来了。

之前使用seer，可是最近发现开始走向臃肿和收费的道路(心累)。然后，QuickLook站出来了！这是一个强大的，开源的，免费的Windows文件预览工具。预览视频，图片，音频，代码，迅速高效，用上你就回不去了…

显示屏亮度自动调节工具

f.lux

整天对着电脑上班工作的同学，想必眼睛每天都要受到来自显示屏深深的恶意(显示屏OS：”这锅我不背”)，相信没人会想到天暗时调节一下屏幕亮度来保护眼睛。

白天还好，如果是在晚上环境光线不强的时候，电脑屏幕依旧很亮，无异于直接对着太阳看，此刻你需要这个软件来自动帮你把屏幕调暗，并自动设置成防蓝光模式，这对懒懒的你是不是一个相当不错的关爱。微软在17年四月份发布的最新版win10创意版镜像加入了该模式，是不是很赞，马克。

PS：该软件也有手机版。

文件同步工具

OneDrive

主要用来工作用，工作上重要的文件放在里面，即刻同步至云端，在家也可以直接访问办公。至于其它的同步工具，百度云？抱歉对它真的不感冒，一些大的不重要的东西可以考虑。工作文件，安全第一，在百度没有任何隐私可言。

有个缺点，国内访问有时会不稳定，不过大多数时候真心好用，放进去自动同步，没有任何多余操作。另外截图自动保存简直是不能太好用，个人对截屏有不可掩盖的好感。

同步笔记工具

evernote 印象笔记

老牌的笔记软件了，大象的脑袋究竟能装多少东西？反正就是记忆好的意思吧，白绿的配色给人很清爽的感觉。一个账号可以同时拥有国内版和国际版，相当于两个账号。有个缺点就是免费账户只允许两个设备之间的登陆和同步，不过也足够了，而且网页版随时随地登陆访问可以弥补一下吧，一直用在生活纪录上。

有道云笔记

工作用同步笔记，主要用来做学习的笔记，知识点，通话记录，调试记录，等。当然最方便的还是云协作，可以协同开发，这里写协议和工作计划相当合适。
